# personal-site

一个基于 `Astro + Markdown` 的中文个人主页/静态博客，目标部署平台为 `GitHub Pages`。

## 技术选型

- `Astro`
- `Markdown / MDX`
- 纯静态构建

## 本地开发

```sh
npm install
npm run dev
```

## 构建

```sh
npm run build
```

- [src/pages/about.astro](D:/研究生工作/个人主页/src/pages/about.astro) 中的个人介绍
