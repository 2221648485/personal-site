这个站点虽然会以随笔和记录为主，但也保留了写技术内容的能力。

比如以后可以整理这些主题：

- 一个项目踩过的坑
- 某个前端或工程化问题的排查过程
- 研究中用到的工具链记录

```ts
export function keepWriting(topic: string) {
	return `继续写下去：${topic}`;
}
```

对个人博客来说，技术内容不一定要很多，但应该足够真实，最好来自自己真正做过的事情。
